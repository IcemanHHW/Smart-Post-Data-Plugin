# Smart-Post-Data-Plugin
This is a plugin to show the data of the post in the header or somewhere else on the page.

## Usage
The plugin has 2 shorcodes:
1. [spd_blog_title]
⋅⋅⋅Place the title of the post on the page.
2. [spd_blog_date]
⋅⋅⋅Place the title of the post on the page.
